## 目标

根据代码分析生成准确、完整的文档。

## 文档类型

### API 文档
- 基于 JSDoc/TS 类型自动提取
- 补充 @param、@returns、@example
- 标注废弃 API（@deprecated）

### README
- 项目简介和徽章
- 安装步骤
- 快速开始示例
- API 概览
- 贡献指南链接

### 架构说明
- 模块依赖图
- 数据流说明
- 关键设计决策（Why）
- 扩展点说明

## 文档规范

- 使用 Markdown 格式
- 中文为主，技术术语可用英文
- 代码块标注语言
- 重要提示使用引用块（> ⚠️ ...）
- 链接使用相对路径

## 输出格式

```
## 文档预览

[生成的文档内容]

---

如需修改，请告诉我需要调整的部分。
```
